Инструкция по использованию

❗️ Инструкция обязательна к прочтению перед использованием.

Параметр Тип Описание
Общие настройки приложения
Exchange ENUM Биржа, на которой будут открываться позиции.
Api Key STRING API-ключ от выбранной биржи.
Api Secret STRING Секретный ключ от выбранной биржи.
Api Passcode STRING Пасс-фраза от выбранной биржи (только для okx.com).
Risk $ FLOAT Риск в $ в позиции.
Is users blacklist enabled BOOL Включение черного списка пользователей.
Users blacklist List[STRING] Черный список пользователей.
Is tickers whitelist enabled BOOL Включение белого списка монет.
Tickers blacklist List[STRING] Белый список тикеров.
Настройка телеграм уведомлений
Bot Token STRING Токен телеграм-бота, создается в @BotFather.
Chat Id INT Ваш чат ID или ID чата, куда будут присылаться уведомления (получается в @getmyid_bot).
Настройка авторизации в админ-панель
Admin Login STRING Логин для входа.
Admin Password STRING Пароль для входа.
Логи
Страница с отображением логов по уровням. Пользователю нужно проверять вкладки ERROR и INFO. Остальное создано для разработчиков.

После первого запуска приложения необходимо заполнить модели настроек, описанные выше. Приложение поддерживает автоматическое обновление. Для обновления необходимо перезапустить сервер или докер-контейнер. Текущая версия приложения указана в оглавлении в верхнем левом углу.
